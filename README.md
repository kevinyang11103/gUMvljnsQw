# 健身房管理系统 java458

## 项目概述

本项目是基于Java语言的SSM框架和SpringBoot开发的健身房管理系统。该系统采用前后端分离的设计，前端使用Vue.js框架，后端采用Java技术栈，旨在提供一套可靠、易用的健身房管理解决方案。

## 技术栈

- 数据库：MySQL
- 服务器：Apache Tomcat
- 编译器：IntelliJ IDEA
- 项目管理工具：Maven
- 前端：Vue.js

## 功能模块

系统包含三个主要角色：管理员、员工、会员，各自有不同的功能权限。

### 管理员

- 个人中心
- 会员管理
- 员工管理
- 会员卡管理
- 会员卡类型管理
- 教练信息管理
- 解聘管理
- 健身项目管理
- 指导项目管理
- 健身器材管理
- 健身活动管理

### 会员

- 个人中心
- 会员卡管理
- 教练信息查看
- 健身项目管理
- 健身器材管理
- 健身活动管理

### 员工

- 个人中心
- 会员卡管理
- 教练信息查看
- 健身项目管理
- 健身器材管理
- 健身活动管理

## 运行环境

- 后端：Java 1.8+
- 前端：Vue.js运行环境
- 数据库：MySQL 5.7+
- 服务器：Apache Tomcat 8.5+

## 部署步骤

1. 环境准备：确保Java和MySQL服务已正确安装。
2. 数据库配置：导入SQL文件以创建数据库和表结构。
3. 后端部署：将编译后的代码部署到Tomcat服务器。
4. 前端部署：将Vue项目构建后的文件部署到Web服务器。

## 目录结构

```
/backend                -- 后端代码目录
    /src                -- 源码目录
        /main           -- 主代码目录
            /java       -- Java源文件
            /resources  -- 资源文件，如配置文件等
    /pom.xml            -- Maven项目配置文件
/frontend               -- 前端代码目录
    /src                -- 源码目录
        /assets         -- 静态资源
        /components     -- Vue组件
        /router         -- 路由配置
    /index.html         -- 主页面
```

## 核心亮点

- 前后端分离设计，方便维护和扩展。
- 完善的权限控制，满足不同角色的操作需求。
- 基于SSM框架和SpringBoot，保证了系统的稳定性和高效性。
